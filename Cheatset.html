Practical GitHub Workflow Activity

Goal: Initialize a new local project, commit an initial file, connect to a new remote GitHub repository, and push the code.

1. Initialize the Local Repository

Execute:

git init

2. Stage and Commit

Add your initial file(s) and commit:

git add .
git commit -m "Initial commit for exam"

3. Connect Remote Repository

Link your local repository to the remote GitHub repository (replace <URL>):

git remote add origin <URL>

4. Push to GitHub

Push your code to the main branch:

git push -u origin main

5. Final Verification

Provide a screenshot of the GitHub repository page showing the successful upload.
